## 把热点变成你的试用清单（可选）

> 复制下面这段 Prompt，补充你的实际情况，再交给 Agent。它不属于上面的四个固定板块。

```text
请基于上面的《每日 GitHub 热点精选》继续分析。

我的角色是：
我当前要完成的任务是：
我的技术熟悉度是：
我的运行环境是：
我本周可以投入的试用时间是：
我最在意的是：隐私 / 成本 / 易用性 / 可扩展性 / 其他

请：
1. 只推荐最适合我的 3—5 个项目，并说明它们与当前任务的具体关系；
2. 对每个项目说明试用成本、必要依赖、主要风险和不适用边界；
3. 给出一个 30 分钟内可完成的最小验证步骤；
4. 区分报告中的已知事实与你根据描述做出的推断；
5. 指出哪些热门项目目前与我无关，可以暂时跳过。
```
